A program célja

A program kontaktok kezelését hajtja végre. A program képes importálni és exportálni .vcard formátumú fájlokat. A programban a tárolt kontaktok megtekinthetők, szerkeszthetők, és törölhetők.

A program használata

A menüvezérelt programot a felhasználó konzolablakon keresztül kezelheti. Az éppen aktív menü opcióit az annak megfelelő szám bevitelével lehet kiválasztani. Ezután megjelenik annak tartalma, vagy ha az is egy menü, akkor annak opciói, amik közül szintén ugyanígy lehet választani.

  1. új felvétele
    1. név
    2. telefonszám
    3. email
    4. cím
    5. foglalkozás
  2. összes megtekintése
    1. kontakt #1
      1. szerkesztés
          (ezt kiválasztva ugyanolyan menüt kapunk, mint amikor újat veszünk fel.)

A felhasználóval történő egyéb kommunikáció

A hibaüzeneteket a konzolra írja ki a program.
Amennyiben a hiba a felhasználó hibás inputjából adódik, a program új bemenetet kér.
Futásidejű probléma esetén a program tájékoztatja erről a felhasználót, majd kilép.
